# Theme

Technical-minimalist analytics interface adapted from the Mosaic Grid Architecture style.

- Paper: `#f7f7f5`
- Panel: `#ffffff`
- Ink: `#17211b`
- Muted ink: `#667069`
- Forest: `#1a3c2b`
- Forest soft: `#dce7df`
- Coral: `#e78468`
- Gold: `#d6ae45`
- Hairline: `rgba(23, 33, 27, 0.14)`
- Sans: Space Grotesk with system fallback.
- Mono: JetBrains Mono with system monospace fallback.
- Flat surfaces, one-pixel borders, two-pixel radii, no gradients and no decorative shadows.
